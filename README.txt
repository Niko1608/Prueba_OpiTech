------------- EXPLICACION FRONTEND -------------
Para realizar el Frontend, cre un nuevo proyecto mediante la consola y empece a trabajar con visual studio code en el. 
agregue un componente tabla y genere los codigo necesario para su visualizacion, asi tambien genere una base de datos Json 
en la cual meti los datos necesarios para importar a la tabla a mostrar. luego de esto extructure la interfaz en HTML agregando 
los complementos necesarios para generar consultas. y le di estilo mediante CSS.
